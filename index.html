A Lightless Labs research project

Agent behavior, counted from the logs up.

Third Thoughts studies AI coding agents by reading the transcripts they leave behind and counting what actually happens — not what agents say they did, and not the one heroic demo that worked on a Tuesday.

The premise

Session logs are where the behavioral truth lives.

Agent systems leave long, messy, implementation-detail-heavy transcripts. People mostly skip over them. We treat them as behavioral evidence — ugly, repetitive, and useful — and try to turn vague hunches into numbers that can survive contact with reality.

Why it matters

Otherwise you just get opinions with nicer typography.

The corpus can show which working patterns correlate with successful sessions, when a run is drifting toward failure before it fully falls over, and whether a new CLI or model rollout quietly made things worse. The project is deliberately half tool and half research write-up.

Monitor over time

Catch regressions across rollouts.

Rollouts of a new CLI or model version can change agent behavior in ways nobody noticed until the bug tracker lit up a week later. Run the same battery of techniques before and after and compare — the corpus is already versioned.

Working patterns

See what successful sessions actually do.

Self-correction tied to evidence-seeking looks like success. Planning without self-correction looks like something headed straight for a user intervention. Those are empirically distinguishable in the transcript.

Pre-failure signals

Flag sessions drifting toward trouble.

Hidden state models find a recognizable low-output behavioral basin that agents enter before users step in to correct. Magnitude is still provisional; the direction replicates across runs.

Install middens

middens is the Rust CLI that does the digging. Point it at a directory of session transcripts and it parses, classifies, runs a 23-technique analytical battery, and exports results you can inspect without having to pretend raw JSONL is a user interface.

From GitHub Releases

Download the tarball for your platform.

Grab the latest tarball from GitHub Releases, unpack it, and run ./middens --help. Homebrew is coming soon, once the release plumbing stops behaving like release plumbing. 

# macOS (Apple Silicon) or Linux
curl -LO https://github.com/Lightless-Labs/third-thoughts/releases/latest/download/middens-<version>-<target>.tar.gz
tar -xzf middens-*.tar.gz
cd middens-*/
./middens run ./corpus/ --all
